hi there! this is Caitlin Maclatte's website.

She can be found on twitter, github and in Canberra, Australia at the Australian National University studying a Bachelor of Advanced Computing. If you like security, Cait also has keybase!

what does cait do?

Cait does adviceformerrin.tumblr.com and some of the posts on catviceforben.tumblr.com.

She's currently the President of the ANU Computer Science Students' Association, having previously been on the committee as a General Representative between September 2013 and September 2014.

Some interesting projects Cait has worked on include:

  • caitessegonal, a plugin written in C++ for video editing software Nuke which is meant to render a tesselating pattern of octagons and squares. It was made for a digital art course in 2014 Semester One. You can check out its development over at caitelartv2619.tumblr.com.
  • caiteshell, a very basic shell program written in C for a programming course in 2014 Semester One. It doesn't produce any seg faults that she knows of, which is something to be proud of.
  • caitecomp3530 learning portfolio, a collection of entries to document my opinions of systems and software engineering for assessment for COMP3530 - Systems Engineering for Software Engineering for 2015 Semester One.
  • caitechLauncher learning portfolio, a collection of entries to document my work each week in my Software Group Projects course in a context of systems and software engineering. Assessment for COMP3100 - Software Group Projects for 2015 Semester One.
  • caitelatte.github.io, this website. Cait is trying for a slightly nicer approach than plaintext and html, though she'll build up the site as her skills progress. If you see anything wrong or that could use improving on this site, please submit a pull request or bug report! :)

Cait is very interested in how easy it is to work her name into the names of her projects, and also delicious food.
